## 运行环境
JRE8 + MySQL5.7（内附5.5版本SQL脚本）  
端口：9999  
使用idea打开工程时需要插件lombok  
topicstu为学生端app源码，采用HbuilderX开发。  

## 部署注意事项
配置文件路径：/src/main/resources/application.properties

## 运行方法
服务器安装好MySQL5.7并执行建表脚本  
安装好JDK1.8（含JRE8）  
安装screen  
在screen下执行java -jar topicselect-1.0-RELEASE.jar
